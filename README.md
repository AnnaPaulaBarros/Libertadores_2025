# Libertadores_2025
Análise do desempenho de Flamengo e Palmeiras: Libertadores 2025
Este projeto desenvolve um ecossistema de análise estatística para comparar o desempenho tático e técnico entre Flamengo e Palmeiras durante a Copa Libertadores 2025. Utilizando dados de performance (SofaScore), a análise foca em transformar métricas brutas em inteligência competitiva para scouts e comissão técnica.
